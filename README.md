# 🏡 BrickWorth+ – ML-Based Real Estate Price Prediction Platform

> An intelligent machine learning platform for predicting residential property prices using historical housing data and a Django-based web application.

## 📌 Overview

BrickWorth+ is a machine learning-powered real estate price prediction platform that helps users estimate property prices based on key housing attributes. The project combines data preprocessing, regression models, and a Django web application to provide fast and accurate price predictions through an intuitive interface.

---

## ✨ Features

- Predict house prices using Machine Learning
- Compare multiple regression algorithms
- Data preprocessing and feature engineering
- User-friendly Django web interface
- Real-time property price estimation
- Model evaluation using regression metrics
- Modular and scalable project architecture

---

## ⚙️ Project Workflow

1. User enters property details.
2. Input is validated and preprocessed.
3. Trained ML model generates predicted price.
4. Django backend returns prediction.
5. Results are displayed instantly on the dashboard.
