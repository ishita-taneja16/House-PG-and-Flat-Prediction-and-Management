# 🏠 BrickWorth+
## Intelligent house, PG and Flat Price Prediction and Management System

BrickWorth+ is a **full-stack Django-based web application** that provides **machine learning–powered real estate price predictions** along with a **verified accommodation discovery system** for houses, PGs, and flats. The platform combines **backend engineering, supervised ML models, and modern UI design** to support informed Buy/Rent decisions.

---

## 🚀 Project Overview

BrickWorth+ predicts property prices using **real-world features** such as location, area, rooms, bathrooms, and storeys. The system includes **admin-verified listings**, **interactive maps**, and **dynamic input validation**, making it reliable and production-oriented.

This project is designed to simulate a **real-world real estate platform**, focusing on **scalability, security, and usability**.

---

## ✨ Key Features

- 🔹 ML-based price prediction for houses, PGs, and flats  
- 🔹 Buy / Rent workflow for users  
- 🔹 Admin-verified property listings  
- 🔹 Interactive maps for Punjab & Himachal (Leaflet.js)  
- 🔹 Dynamic feature-based input validation  
- 🔹 Role-based access control (Admin / Owner / User)  
- 🔹 AI chatbot for user assistance  
- 🔹 Modern, user-centric UI design  

---

## 🧠 Machine Learning Pipeline

- Data preprocessing and cleaning  
- Feature engineering (area, rooms, bathrooms, storeys, location)  
- Supervised learning models:
  - Linear Regression  
  - Random Forest  
  - XGBoost  
- Model persistence using Pickle (`.pkl`)  
- Real-time predictions via Django backend

## 🧩 System Architecture

1. User inputs data via frontend forms or maps  
2. Requests are sent to Django backend  
3. Backend validates input and processes logic  
4. ML models generate price predictions  
5. Results are returned dynamically to the UI  
6. Admin approval ensures only verified listings are displayed
